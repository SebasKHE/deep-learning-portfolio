# Module 1: Neural Networks and Deep Learning

This module contains foundational implementations of neural networks built from scratch, demonstrating core concepts of deep learning.

## üìÇ Projects

### 1. Building Deep Neural Networks Step-by-Step
**File**: `Building_your_Deep_Neural_Network_Step_by_Step.ipynb`

Complete implementation of an L-layer deep neural network from scratch, including:
- Forward propagation with multiple activation functions (ReLU, Sigmoid)
- Backward propagation using chain rule
- Parameter initialization and updates
- Modular design for scalable network architectures

**Key Achievements**:
- Built reusable functions for any L-layer network configuration
- Implemented efficient vectorized operations
- Demonstrated understanding of gradient flow through deep networks

---

### 2. Deep Neural Network Application
**File**: `Deep Neural Network - Application.ipynb`

Binary image classification project using the deep neural network framework.

**Implementation Highlights**:
- Applied L-layer network to real image data
- Preprocessed and normalized image datasets
- Achieved high accuracy on binary classification task
- Analyzed model performance and decision boundaries

---

### 3. Planar Data Classification
**File**: `Planar_data_classification_with_one_hidden_layer.ipynb`

Single hidden layer neural network for classifying non-linear planar data.

**Technical Focus**:
- Demonstrated neural networks' ability to learn non-linear decision boundaries
- Implemented complete training loop from scratch
- Visualized decision boundaries and model predictions
- Compared performance with logistic regression baseline

---

### 4. Python Basics with NumPy
**File**: `Python_Basics_with_Numpy.ipynb`

Foundation for efficient deep learning computations using NumPy.

**Core Concepts**:
- Vectorization for performance optimization
- Broadcasting and array manipulation
- Mathematical operations for neural networks
- Sigmoid function and gradient computation

- **Programming**: Vectorization, modular code design, NumPy proficiency
- **Deep Learning Concepts**: Activation functions, cost functions, parameter updates
- **Problem Solving**: Building networks from first principles
