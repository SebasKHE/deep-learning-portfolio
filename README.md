### [Module 1: Neural Networks and Deep Learning](./1%20Neural%20networks%20and%20deep%20learning)

Foundational implementations of neural networks from scratch.

- **Building Deep Neural Networks Step-by-Step** - Complete implementation of L-layer neural networks with forward/backward propagation
- **Deep Neural Network Application** - Binary image classification using deep networks
- **Planar Data Classification** - Single hidden layer network for non-linear decision boundaries

**Key Skills**: Forward/backward propagation, activation functions, gradient descent, vectorization

### [Module 2: Optimization & Regularization](./2%20Initialziation,%20Optimization,%20hiperparameter%20tunning%20regularization)

Advanced training techniques for improving model performance and generalization.

- **Weight Initialization** - Comparing zero, random, and He initialization strategies
- **Optimization Methods** - Implementation of Mini-batch GD, Momentum, RMSprop, and Adam
- **Regularization Techniques** - L2 regularization and Dropout for preventing overfitting
- **Gradient Checking** - Numerical verification of backpropagation implementations
- **TensorFlow Introduction** - Building neural networks with TensorFlow framework

**Key Skills**: Hyperparameter tuning, optimization algorithms, regularization, TensorFlow/Keras

### [Module 3: Convolutional Neural Networks](./3%20Convolutional%20Networks)

Computer vision projects using state-of-the-art CNN architectures.

- **CNN Fundamentals** - Building convolutional layers, pooling, and full CNN models from scratch
- **ResNet Implementation** - Deep residual networks with skip connections
- **Transfer Learning with MobileNet** - Fine-tuning pre-trained models for custom classification
- **Object Detection (YOLO)** - Autonomous driving car detection system
- **Face Recognition** - Siamese network for face verification and recognition
- **Image Segmentation (U-Net)** - Semantic segmentation for medical imaging
- **Neural Style Transfer** - Artistic style transfer using VGG network
- **CNN Applications** - Sign language recognition with custom CNN

**Key Skills**: CNNs, transfer learning, object detection, image segmentation, ResNets, MobileNet, U-Net

### [Module 4: Sequence Models](./4%20Sequence%20Models)

Recurrent neural networks for sequential data processing.

**Key Skills**: RNNs, LSTMs, sequence modeling, text generation, music composition

## üìã Setup Instructions

### Prerequisites

- Python 3.7+
- pip or conda package manager

### Installation

1. Clone this repository:
```bash
git clone https://github.com/YOUR_USERNAME/deep-learning-portfolio.git
cd deep-learning-portfolio
```

2. Install required dependencies:
```bash
pip install -r requirements.txt
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```

4. Navigate to any module folder and open the desired notebook.
